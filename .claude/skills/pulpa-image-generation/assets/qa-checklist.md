# QA checklist for pulpa images

Use this checklist before accepting any generated or edited image.

## Generation integrity

- [ ] The image was generated/designed for the correct fruit from the start.
- [ ] The output is one isolated product asset, not an ad/poster/flyer/card/layout.
- [ ] It does not look like another fruit recolored.
- [ ] The pouch texture, pulp color, and fruit garnish match the fruit profile.
- [ ] The output is photorealistic enough for a premium catalog.

## Presentation structure

- [ ] 120g is a slim narrow vertical pouch, visibly smaller and narrower than 300g.
- [ ] 300g is a medium vertical pouch, wider and fuller than 120g.
- [ ] 1000g is a wider, more robust vertical institutional pouch than 300g.
- [ ] Relative scale order is correct: 120g < 300g < 1000g in pouch width and visual mass.
- [ ] On a normalized 1000x1000 canvas, 120g pouch-only box is about 260-310px wide and 540-600px tall; full group is about 380-450px wide and 600-650px tall.
- [ ] On a normalized 1000x1000 canvas, 300g pouch-only box is about 390-450px wide and 660-730px tall; full group is about 560-640px wide and 730-800px tall.
- [ ] On a normalized 1000x1000 canvas, 1000g pouch-only box is about 510-590px wide and 740-820px tall; full group is about 680-780px wide and 820-880px tall.
- [ ] The package is a flat heat-sealed frozen pulp sachet, not a stand-up zipper/doypack/retail pouch.
- [ ] The presentation belongs to the same vertical family while remaining distinguishable by pouch width, label text, and controlled scale.
- [ ] The generated image uses the approved canonical reference for that presentation as structure only.
- [ ] The pouch geometry matches the same-presentation approved reference across flavors.

## Packaging quality

- [ ] The pouch looks full, fresh, and naturally tense.
- [ ] It does not look empty, sagging, over-inflated, or crushed.
- [ ] It is not exaggerated, balloon-like, or shaped like a swollen pillow.
- [ ] The pouch has an extremely subtle 1 degree rotation toward the viewer's left, with the right side edge slightly more visible, while keeping the label readable.
- [ ] The image is not perfectly frontal and not a strong 3/4 perspective.
- [ ] Side seals are visible but soft, physical, and subordinate.
- [ ] 120g top and side seal plastic is clean, thin, translucent, and lightly visible; not ribbed, striped, cloudy, dirty, thick, or dominant.
- [ ] Wrinkles and highlights reinforce real flexible plastic.
- [ ] The label is integrated into the pouch perspective/material.
- [ ] The label contains only `Pulpa de Fruta Natural` and the flavor name; no weight text, numbers, grams, claims, badges, or extra copy.

## Composition

- [ ] The product group is optically centered.
- [ ] Fruit garnish supports the pouch and does not dominate.
- [ ] No product element touches canvas edges.
- [ ] The product is not oversized on the canvas and preserves the target safe margins for its presentation.
- [ ] The background is true transparent alpha, or plain white if alpha is unsupported; no rendered checkerboard grid, fake transparency pattern, dark hero background, or designed scene.
- [ ] The label is readable at catalog-card size.
- [ ] The image feels consistent with approved patterns for the same presentation.
- [ ] Only pulp color, garnish fruit, and flavor text changed compared with same-presentation variants.

## Final export

- [ ] Final master is `1000x1000px`.
- [ ] Background is transparent.
- [ ] PNG alpha is clean.
- [ ] Web export can be generated from approved master.
